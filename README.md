# VendmoTech_Network_Traffic_Analysis
A Wireshark-based network traffic analysis simulating a live SOC incident at Vendmo Tech. Detects C2 beaconing, data exfiltration &amp; port scanning across a 2.3GB PCAP. Includes 8 findings, 10 IOCs, MITRE ATT&amp;CK v14 mapping &amp; attack timeline. Blue Team / SOC portfolio project.
